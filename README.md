🌊 Sea Level Predictor
📌 Project Overview

This project analyzes global average sea level changes from 1880 to 2014 using data from the US Environmental Protection Agency.

Using Python, Pandas, Matplotlib, and Scipy, we:

Visualize historical sea level data

Calculate lines of best fit

Predict sea level rise through the year 2050

🛠 Technologies Used

Python 3

Pandas

Matplotlib

Scipy

Git & GitHub

📊 Project Tasks

✔ Import dataset using Pandas
✔ Create scatter plot
✔ Compute regression line (1880–2050)
✔ Compute regression line (2000–2050)
✔ Predict sea level rise in 2050
✔ Label and title graph
✔ Save output image

📁 Dataset Source

Global Average Absolute Sea Level Change, 1880–2014
Source: US Environmental Protection Agency
Data from CSIRO (2015) and NOAA (2015)
